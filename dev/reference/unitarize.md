# Unitarize the singular values in a `CompartmentCall` or `MultiCompartmapCall`

Unitarize the singular values in a `CompartmentCall` or
`MultiCompartmapCall`

## Usage

``` r
unitarize(x, medianCenter = TRUE)
```

## Arguments

- x:

  A `CompartmentCall` object

- medianCenter:

  Whether to center the singular values on their median
