# Check that the input SummarizedExperiment object has the right assays

Check that the input SummarizedExperiment object has the right assays

## Usage

``` r
verifyAssayNames(se, assay)
```

## Arguments

- se:

  Input SummarizedExperiment object

- assay:

  The assay type

## Value

Error if the right assay type is not present, NULL if it is
